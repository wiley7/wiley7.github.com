一致性哈希算法

使用场景

负载均衡有几种不同的算法(分布式环境中的负载均衡策略)：

  • 轮询
  • 随机选择
  • 优先级
    • 本机优先
    • 本机房优先
    • 权重相关的轮询
  • 哈希

一般情况下，在使用分布式缓存的时候，前面几种都不是很适合。缓存的作用就是为了相同的内容对象需要落在同一台机器上，这样缓存中的数据才能派上用处，缓存命中率才会比较高。缓存算法，可以是hash(A) % N，A为对象，N为节点个数。
