php中使用redis的序列化

在php中，使用redis，经常会涉及是否使用序列化。经查源码发现，redis自身不会对数据进行序列化，而是由PHP的redis扩展phpredis进行的。它提供三种方式：

  • Redis::SERIALIZER_NONE 不序列化
  • Redis::SERIALIZER_PHP 使用PHP自身的变量序列化方式
  • Redis::SERIALIZER_IGBINARY 使用二进制

PHP实现依赖注入

依赖注入（控制反转）是什么

依赖注入 ，旨在降低代码耦合度。Wikipedia中这样描述

Class A中用到了Class B的对象b，一般情况下，需要在A的代码中显式的new一个B的对象。
采用依赖注入技术之后，A的代码只需要定义一个私有的B对象，不需要直接new来获得这个对象，
而是通过相关的容器控制程序来将B对象在
外部new出来并注入到A类里的引用中。
而具体获取的方法、对象被获取时的状态由配置文件（如XML）来指定。
